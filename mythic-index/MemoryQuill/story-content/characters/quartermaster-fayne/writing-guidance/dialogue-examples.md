# Quartermaster Fayne - Dialogue Examples

**Quick Reference:** Fayne is analytical (12.8 words/line), data-driven, uses "Pattern shows..." (ONLY Fayne), specific numbers always present, never vague estimates. Methodical, evidence-based observations.

---

## Authentic Examples (90-100)

### Example 1: Pattern Analysis
**Dialogue:** "Pattern shows 3.2-minute intervals between assault waves. Distribution: 60% northern approach, 40% eastern flank. Suggests coordinated timing, not random."

**Score:** 98/100 - Signature "Pattern shows," specific numbers (3.2, 60%, 40%), analysis

### Example 2: Data-Driven Observation
**Dialogue:** "Fourteen instances of this behavior pattern. Average duration: 47 seconds. Confidence in prediction: 87%."

**Score:** 96/100 - Specific counts, averages, confidence percentages

### Example 3: Evidence-Based Assessment
**Dialogue:** "Supply consumption suggests 42 personnel, not 35. The pattern doesn't align with reported numbers."

**Score:** 95/100 - Pattern + specific numbers + analysis

---

## Usage Guidelines

**When to Load:** Fayne analysis, tactical assessment, pattern recognition scenes

**Quick Voice Test:** Are there specific numbers? Does it include "Pattern shows/suggests"? Is analysis present? If yes, likely Fayne.
