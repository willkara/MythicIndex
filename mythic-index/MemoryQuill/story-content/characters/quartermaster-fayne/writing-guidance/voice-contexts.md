# Quartermaster Fayne - Voice by Context

---

## Context Distribution

- **Analysis/Reporting:** 80% (pattern observation)
- **Urgent/Combat:** 15% (quick data assessment)
- **Rare Uncertainty:** 5% (when data insufficient)

---

## Analysis/Reporting Context (80%)

**Words/line:** 13.2 avg
**Key Markers:** Specific numbers, "Pattern shows/suggests," methodical structure

### Characteristics

- Full analytical statements with specific data
- Multiple data points per observation
- Structured: pattern, distribution, percentage, implication

### Example

"Pattern shows 3.2-minute intervals. Distribution: 60% northern, 40% eastern. Suggests coordinated assault, not random."

---

## Urgent/Combat Context (15%)

**Words/line:** 10.1 avg (shorter for speed)
**Key Markers:** Quick data, still specific numbers, rapid assessment

### Characteristics

- Data compressed for speed
- Still specific (no vague guesses)
- Pattern + implication: result-focused

### Example

"Pattern: 47-second assault waves. High confidence: coordinated. Recommend defensive positioning adjustment."

---

## Rare Uncertainty Context (5%)

**Words/line:** 11.4 avg
**Key Markers:** "Insufficient data," "pattern unclear," acknowledging limitation

### Characteristics

- Rare moments acknowledging data gaps
- Never vague ("maybe"), only "insufficient data"
- Seeks additional observation

### Example

"Insufficient data for pattern confirmation. Need 3-4 more observation cycles for accurate assessment."

---
